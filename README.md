# CPU Scheduling Dashboard

## Project Overview
This project is an interactive web-based CPU Scheduling Simulator that implements:
- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- Round Robin

## Features
- Dynamic process input
- Algorithm selection dropdown
- Time quantum input (for Round Robin)
- Performance metrics:
  - Waiting Time (WT)
  - Turnaround Time (TAT)
- Interactive charts using Chart.js

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

##  Output
- Table showing CT, WT, TAT
- Graphs for:
  - Waiting Time
  - Turnaround Time
